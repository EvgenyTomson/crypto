## Ипользованные инструменты:

- React
- TypeScript
- SASS
- Vite

### Установка и запуск

Установить Node.js  
Node Version Manager

```bash
nvm install
```

Fast Node Manager

```bash
fnm install
fnm use
```

Установить зависимости:

```bash
npm install
```

Запустить проложение:

```bash
npm run dev
```
